# 🚢 Titanic Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on the Titanic dataset.
The goal is to understand the dataset using **statistics and visualizations**, and identify patterns, trends, and relationships between features.

---

## 🎯 Objectives

* Understand the structure of the dataset
* Perform descriptive statistical analysis
* Visualize data distributions and relationships
* Identify patterns affecting survival

---

## 🛠 Tools & Technologies Used

* Python 🐍
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📊 Dataset

The dataset used is the **Titanic dataset**, which contains information about passengers such as age, gender, ticket class, fare, and survival status.

---

## 🔍 Steps Performed

### 1. Data Loading

* Imported dataset using Pandas

### 2. Data Understanding

* Used `.head()`, `.info()`, `.describe()`

### 3. Data Cleaning

* Checked for missing values

### 4. Data Visualization

* Histograms for distribution
* Boxplots for outliers
* Countplots for categorical features
* Heatmap for correlation
* Pairplot for relationships

---

## 📈 Key Insights

* 👩 Female passengers had a higher survival rate than males
* 🏆 Passengers in **1st class** had higher survival chances
* 👶 Younger passengers had slightly better survival rates
* 💰 Fare is positively correlated with survival
* 📉 Missing values found in Age column

---

## 📌 Conclusion

EDA helped in understanding important patterns in the dataset, which can be useful for building machine learning models for survival prediction.

## ❓ Interview Questions (with Answers)

### 1. What is EDA?

EDA (Exploratory Data Analysis) is the process of analyzing datasets using statistics and visualization techniques to understand patterns before building models.

### 2. What is correlation?

Correlation shows the relationship between two variables. It ranges from -1 to +1.

### 3. What is multicollinearity?

When two or more features are highly correlated, it is called multicollinearity.

### 4. How do you detect skewness?

Using histograms or comparing mean and median values.

### 5. Role of visualization?

Visualization helps in easily understanding patterns, trends, and anomalies.
